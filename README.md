Attendance Tracking System
Project Overview

This project is a web-based Attendance Tracking System designed to monitor, manage, and analyze employee or student attendance efficiently. It allows users to view, update, and track attendance records, along with summary statistics for better management.

Features

User Profile Management

Users can view and update personal details like name, email, department, phone, and location.

Profile updates are saved in real-time with notifications.

Attendance Recording

Users can mark attendance as Present, Absent, Late, or Half-Day.

Records are stored and tracked over time.

Attendance History

View historical attendance records with filters by date and status.

Pagination for easy browsing of large datasets.

Summary Statistics

Displays key statistics such as total days present, absent, late arrivals, half-days, and total hours worked.

Responsive UI

Built with React, Tailwind CSS, and reusable components.

Real-time updates using Zustand for state management.

Tech Stack

Frontend: React, Tailwind CSS

State Management: Zustand

Backend: Node.js + Express (API integration)

Database: MongoDB (or any preferred DB)

Icons & UI: Lucide React, custom components

Installation / Setup

Clone the repository:

git clone <your-repo-link>
cd <repo-folder>


Install dependencies:

npm install


Start the development server:

npm start


Backend API:
Ensure your backend API (Node.js + Express) is running and connected to your database.

Usage

Profile Management

Navigate to the Profile page to view or edit personal details.

Mark Attendance

Check-in/check-out or update status using the dashboard interface.

View History

Access the Attendance History page to see past records.

Use search and filters to find specific records.

Statistics

Monitor your attendance summary on the dashboard with present, absent, late, half-day counts, and total hours worked.
